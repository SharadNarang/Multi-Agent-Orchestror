 HOW TO USE THE SYSTEM


 Step 1: Set Your OpenAI API Key
   $env:OPENAI_API_KEY = 'your-actual-api-key-here'
   (Get-Content backend\.env) -replace 'OPENAI_API_KEY=.*', \"OPENAI_API_KEY=$env:OPENAI_API_KEY\" | Set-Content backend\.env

 Step 2: Restart Services (after setting API key)
   Get-Job | Stop-Job; Get-Job | Remove-Job
   Then restart using the commands in Step 3

 Step 3: Access API Documentation
   Main API: http://localhost:8000/docs
   A2A Agent: http://localhost:8001/docs
   API Agent: http://localhost:8002/docs



 EXAMPLE: CREATE AND EXECUTE A TASK


# Create a task
$taskBody = '{"description":"Research AI agents and summarize","user_id":"user1"}'
$response = Invoke-WebRequest -Uri \"http://localhost:8000/api/tasks\" -Method Post -Body $taskBody -ContentType \"application/json\" -UseBasicParsing
$task = $response.Content | ConvertFrom-Json

# Check task status
$taskId = $task.task_id
Invoke-WebRequest -Uri \"http://localhost:8000/api/tasks/$taskId\" -UseBasicParsing | Select-Object -ExpandProperty Content | ConvertFrom-Json
